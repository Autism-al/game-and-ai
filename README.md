## 运行环境
- 操作系统：Win10
- Python版本:3.8以上 
- 依赖参照requirements.txt

> pip install -r requirements.txt

## 编译方法以及使用方法
- 安装完依赖后直接运行图片华容道中的huarongdao.py
> python huarongdao.py

- AI中的A_star.py 为AI大比拼包含算法程序，直接运行即可获取当前排名、未做题目并解题
> python A_star.py

- UI中存放的是原型设计的HTML文件
